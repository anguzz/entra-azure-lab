# entra-azure-lab

Personal Microsoft cloud lab for testing and documenting Entra ID, Azure, Microsoft 365, and Exchange Online configurations.

This repo is used as a central place to keep notes, setup steps, screenshots, scripts, and configuration references for my personal tenant/lab environment.


## Purpose

The goal of this lab is to build a small Microsoft cloud environment where I can safely test identity, access control, email security, licensing, and automation without touching a production tenant

This lab gives me a place to practice real Microsoft cloud administration tasks such as:

- Setting up a custom domain in Microsoft 365
- Creating users and assigning licenses
- Configuring Exchange Online mailboxes
- Testing SPF, DKIM, and DMARC
- Working with Entra ID roles and admin permissions
- Testing Conditional Access and identity security features
- Testing new Intune and device management features
- Building PowerShell / Graph API automation
- Documenting changes for future reference
